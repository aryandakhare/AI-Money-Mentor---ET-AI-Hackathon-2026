## 📊 Demo Video Specifications

### Recording Requirements
- **Tool:** OBS Studio (free), Loom, or ScreenFlow
- **Resolution:** 1920x1080 (1080p)
- **Frame Rate:** 30fps
- **Duration:** 3 minutes max
- **Format:** MP4 (H.264)

### Audio
- **Voiceover:** Clear, professional English
- **Background Music:** 
  - Royalty-free (Pexels, Pixabay, YouTube Audio Library)
  - Uplifting, modern vibe (electronic or corporate)
  - 50% volume under voiceover
- **Sound Effects:** 
  - Button clicks
  - Smooth transitions
  - Success "ding" on calculations

### Visual Elements
- **Colors:** 
  - Primary: Purple gradient (#667eea → #764ba2)
  - Accent: Green (#4CAF50) for positive metrics
  - Neutral: Gray for disabled states
- **Typography:** 
  - Font: Segoe UI, Roboto, or system sans-serif
  - Size: 16-24px for readability
- **Animations:**
  - Smooth fade-in for screens
  - Animated score bars (0 → 100 fill)
  - Card slide-in for recommendations
  - Subtle bounce on hover

### Transitions
```
00:00 → 00:30: Fade-in + zoom on problem statement
00:30 → 01:00: Smooth scroll down landing page
01:00 → 01:30: Gentle fade to Money Health Score
01:30 → 02:00: Slide-in for FIRE results cards
02:00 → 02:30: Chat message bubble animation
02:30 → 03:00: Zoom out to landing + fade to black
```

### Subtitles (Optional)
- Add captions for accessibility
- Highlight key metrics (₹60K tax savings, etc.)
- Sync with voiceover

## 🎬 Recording Checklist

**Before Recording:**
- [ ] Quit all background apps (Slack, Notifications, etc.)
- [ ] Close unnecessary tabs/windows
- [ ] Set screen brightness to 70%
- [ ] Test microphone (clear, no background noise)
- [ ] Test internet connection (stable for API calls)

**Recording Tips:**
- [ ] Do multiple takes (use best one)
- [ ] Slow down mouse movements for clarity
- [ ] Pause 1-2 seconds on key screens
- [ ] Leave white space for text overlays
- [ ] Save raw footage before editing

**Post-Production:**
- [ ] Add voiceover (Audacity or Adobe Audition)
- [ ] Add background music (fade in/out)
- [ ] Color grade (match brand colors)
- [ ] Add text overlays for key metrics
- [ ] Add captions/subtitles (SRT format)
